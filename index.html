Secrets

A small app written in Python, Gtk4 and Libadwaita that can generate passwords. Users are able to choose what characters they want to use. The algorithm probably needs some improvement. The app is NOT intended to replace your password manager but as a backup solution if the integrated password generator in, for example, Firefox doesn’t recognize the password field or you need to generate a password for an app where you can’t use your password manager.
